Onboard PUC sensor. The device runs warm, so temperature reads above ambient — watch the trend, not the absolute. Live = the most recent ~10h the API returns; History = your accumulated local record (export/import to keep it across machines).

Data: BirdWeather station 27598 via the public GraphQL API (/graphql) for time‑series, detections and onboard sensors, and the REST API (/api/v1) for species imagery. Detections are automated BirdNET acoustic identifications and may include false positives. Confidence/scatter use a sample of the most recent detections in range. The temperature reading is from the station's own device sensor and runs warmer than ambient. Times shown in the station's local zone. The API only returns the most recent ~10h of sensor readings, so each load is merged into a local history (stored in your browser; export/import as JSON to keep it).
